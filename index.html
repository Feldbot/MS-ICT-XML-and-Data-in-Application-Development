My final project uses the Journal Article Tagging Suite (JATS) XML vocabulary to tag two scholarly journal articles with semantic information used in academic publishing. A single XSL transform file extracts data from each article's XML file and renders the output as HTML. XSL calculations are used to sum the number of tables and graphics in each article.
